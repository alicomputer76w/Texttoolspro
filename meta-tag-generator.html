Essential Meta Tags Explained

Basic SEO Tags:

  • Title Tag: Appears in search results and browser tabs
  • Meta Description: Summary shown in search results
  • Meta Keywords: Less important now, but still used by some engines
  • Robots Tag: Controls how search engines crawl your page

Open Graph Tags:

  • og:title: Title when shared on Facebook, LinkedIn
  • og:description: Description for social media sharing
  • og:image: Image displayed when shared
  • og:url: Canonical URL of the page

Twitter Card Tags:

  • twitter:card: Type of Twitter card (summary, large image)
  • twitter:title: Title for Twitter sharing
  • twitter:description: Description for Twitter
  • twitter:image: Image for Twitter cards

Meta Tag Best Practices

  • Title Length: Keep titles between 50-60 characters
  • Description Length: Aim for 150-160 characters
  • Unique Content: Each page should have unique meta tags
  • Include Keywords: Use target keywords naturally
  • Compelling Copy: Write for users, not just search engines
  • Image Optimization: Use high-quality images (1200x630px recommended)

Common Meta Tag Mistakes

  • Duplicate Tags: Using the same tags across multiple pages
  • Keyword Stuffing: Overloading with keywords
  • Too Long/Short: Not following character limits
  • Missing Images: Not including social sharing images
  • Outdated Information: Not updating tags when content changes

Advanced Meta Tags

Additional Useful Tags:

  • author: Specify content author
  • viewport: Control mobile display
  • canonical: Prevent duplicate content issues
  • robots: Control crawling and indexing
  • theme-color: Set browser theme color

Mobile Optimization

  • Always include viewport meta tag
  • Ensure images work on mobile devices
  • Test social sharing on mobile platforms
  • Consider mobile-specific descriptions if needed
